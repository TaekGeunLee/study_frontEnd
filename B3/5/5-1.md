```javascript
var x, y, z;

var arr1 = [25, 98, 303];

[x, y, z] = arr1;

console.log("x : " + x); // 25
console.log("y : " + y); // 98
console.log("z : " + z); // 303
```

배열 arr1를 디스트럭칭(distructing) 하여 내부의 각 원소들을 
변수 x, y, z에 적용했다.
디스트럭칭을 사용할 떄에는 좌변에는 변수, 우변에는 숫자들의 배열을 넣어둔다.

```javascript
// 예제 1
var a, b, c;
[a, b, c] = [15, 17];
console.log(a); // 15
console.log(c); // undefined

// 예제 2
var a, b;

[a, b, ...c] = [2, 3, 4, 200];

console.log(a);
console.log(c);
```
